Test checklists

Separate QA checklists for the iOS App Store app and the macOS app.

BigSky Planner iOS

Use this for iPhone and iPad release checks.

Download iOS checklist PDF

  • Verify first-start privacy/internet page.
  • Verify Locations, Telescope, Targets, Planner, Weather, Set Up, and Help.
  • Verify iOS screenshots, App Store assets, icon, and version/build.

Astronomy Observation Planner macOS

Use this for desktop release and App Store review checks.

  • Verify version shows macOS 1.1 build 9.
  • Verify first-launch privacy/internet/location agreements, trial splash, and retained setup state.
  • Verify locations fit, telescope/accessory add buttons, telescope/accessory defaulting, target database pop-up, blocked sky areas, and plan export.
  • Verify universal macOS archive behavior on Apple Silicon and Intel where available.
